# README for current design of Budgeting Blast! #

* `applab/applab.js` - The applab library. Contains javascript that applab uses to run the app.

* `applab/applab.css` - Default CSS styling for app.

* `index.html` - html file that designed in applab. Contains all the buttons, text inputs, and other controls for app.

* `style.css` - Custom styling for each control. Colors, positions, dimensions, and other control properties.

* `code.js` - Code for app behavior.

* `assets` - Directory with all the asset files (pictures and sounds).
